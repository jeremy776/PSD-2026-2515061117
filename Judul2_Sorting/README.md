# Penyusun Harga Barang

Program Penyusun Harga Barang ini dibuat untuk membantu mengurutkan harga barang yang ingin dibeli. Jadi misalnya kita punya beberapa barang dengan harga yang beda beda, program ini bisa bantu menyusun dari yang paling murah sampai yang paling mahal.

Di program ini saya menggunakan metode pengurutan yaitu Insertion Sort. Cara kerjanya kurang lebih kayak kita nyusun kartu, jadi setiap angka diambil lalu dimasukin ke posisi yang tepat sampai akhirnya semua data jadi urut.

Setelah selesai, program bakal nampilin daftar harga yang sudah diurutkan (dari termurah sampe termahal). Formatnya saya buat menjadi rupiah supaya lebih enak dan mudah untuk dibaca
